# Python Sports Dictionary Tool

A file-based Python program for reading, updating, and analyzing athlete yardage statistics from multiple .txt files.

## Overview
A homework-style Python project where I designed a command-line tool to load multiple text files, parse player data, and return statistics using a dictionary-based structure.

## Tools and Skills
- Python
- File I/O
- Dictionaries and parsing
- Error handling
- Modular scripting

## Key Contributions
- Implemented multi-file parsing with try/except handling.
- Designed a dictionary structure to store and query player stats.
- Added user commands and validation logic.
